# Face Mask Detection

A Deep Learning based Face Mask Detection system using TensorFlow, Keras, and OpenCV.

## Features

- Real-time face mask detection
- CNN-based classification
- Webcam support
- Mask / No Mask prediction

## Technologies Used

- Python
- TensorFlow
- Keras
- OpenCV
- NumPy
